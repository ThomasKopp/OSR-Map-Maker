# OSR Map Maker: Vorschlaege fuer naechste Funktionen und Symbole

## Funktionsverbesserungen

### Zeichen- und Bearbeitungsworkflow

- [x] Freie Polygonpunkte nachtraeglich einzeln verschieben, einfuegen und loeschen.
- [x] Linien und Polygone optional als Kurven, gestrichelte Linien oder gepunktete Linien darstellen.
- [x] Pfeilspitzen fuer Linien anbieten, z. B. einseitig, beidseitig oder keine.
- [x] Formen mit Vordergrund-/Hintergrundfarbe, Deckkraft und Linienstil speichern.
- [x] Mehrere ausgewaehlte Objekte gemeinsam ausrichten: links, rechts, oben, unten, horizontal zentriert, vertikal zentriert.
- [x] Mehrere ausgewaehlte Objekte gleichmaessig verteilen.
- [x] Objekte per numerischem Panel rotieren, nicht nur Symbole.
- [x] Kopieren/Einfuegen ueber Zwischenablage ergaenzen.
- [x] Objektvorlagen speichern, z. B. "roter Gefahrenkreis", "geheime GM-Notiz", "dicke Grenze".
- [x] Schnellzugriff fuer zuletzt genutzte Werkzeuge und Symbole einbauen.
- [x] Kontextmenue um "Ebene wechseln", "Als Vorlage speichern" und "In Auswahl einpassen" erweitern.

### Kartenstruktur und Navigation

- [x] Mehrere Karten/Etagen pro Projekt unterstuetzen.
- [x] Treppen, Portale oder Marker zwischen Etagen verknuepfen.
- [x] Kartenbereiche als benannte Zonen definieren, z. B. "Koboldbau", "Unterstadt", "Tempel".
- [x] Objektlisten-Panel mit Suche, Filter nach Typ/Ebene und direkter Auswahl ergaenzen.
- [x] Navigator fuer grosse Karten mit Sprungmarken und gespeicherten Ansichten.
- [x] Optionales Koordinatensystem mit Buchstaben/Zahlen am Kartenrand anzeigen.
- [x] Messwerkzeug fuer Distanz, Flaeche und Weglaenge einbauen.
- [x] Lineal-Overlay mit aktueller Raster- und Realwelt-Distanz anzeigen.

### Raum- und Kampagnenfunktionen

- [x] Raumdaten als eigenes Formular mit Tabs fuer Beschreibung, Gegner, Schatz, Fallen und Vorlesetext ausbauen.
- [x] Raumnummern automatisch neu sortieren und freie Nummernluecken erkennen.
- [x] Raumstatus speichern: unentdeckt, entdeckt, gepluendert, gesichert, gefaehrlich.
- [x] Begegnungstabellen pro Karte oder Zone hinterlegen.
- [x] Schatz-/Loot-Tabellen pro Raum hinterlegen.
- [x] Geruechte, Hinweise und Geheimnisse an Raeume oder Symbole binden.
- [x] Spielerhandout-Text getrennt von GM-Notizen pflegen.
- [x] Markdown-Raumreport mit Inhaltsverzeichnis und Symbollegende erweitern.
- [x] Export als GM-Booklet mit Karte plus Raumtexten vorbereiten.

### Prozedurale Tools

- [x] Dungeon-Generator mit Parametern fuer Dichte, Raumgroessen, Rundgaenge und Sackgassen erweitern.
- [x] Generator fuer natuerliche Hoehlen mit Tunneln, Kammern und Wasserlaeufen.
- [x] Automatische Tueren nach Typ platzieren: normal, geheim, verriegelt, Falltuer.
- [x] Automatische Wandstaerken und Wandtypen pro Bereich.
- [x] Zufallsinhalt fuer Raeume erzeugen: leer, Monster, Falle, Schatz, Spezial.
- [x] Zufaellige Patrouillenrouten generieren und als Linien anzeigen.
- [x] Verbindungsvorschlaege interaktiv bestaetigen oder verwerfen.
- [x] "Dungeon aus Stichworten" als Assistent vorbereiten, z. B. Thema, Fraktion, Gefahr.

### Export und VTT

- [ ] Exportprofile speichern, z. B. "Print A4", "Foundry Player", "Roll20 GM", "Fantasygrounds".
- [ ] Batch-Export fuer GM-Version, Player-Version und Gridless-Version in einem Schritt.
- [ ] Exportbereich per benanntem Rahmen speichern.
- [ ] Transparenz/Deckkraft fuer einzelne Ebenen im Export beachten.
- [ ] Foundry-VTT-Szenendaten als JSON exportieren, inklusive Grid, Walls, Doors und Lights.
- [ ] Roll20-kompatible Page-Daten vorbereiten, soweit moeglich.
- [ ] Separater Export fuer Legende und Raumreport.
- [ ] SVG-Export um editierbare Gruppen-IDs je Objekt erweitern.

### Usability und Sicherheit

- [ ] Autosave mit Wiederherstellungsdialog beim Start.
- [ ] Projektvalidierung mit konkreten Warnungen statt nur Abbruch.
- [ ] "Unsaved changes"-Warnung beim Schliessen, Laden und Neu-Anlegen.
- [ ] Undo-Historie im UI sichtbar machen.
- [ ] Tastaturkuerzel live neu binden, ohne Neustart.
- [ ] Farbpaletten pro Projekt speichern.
- [ ] Helles/dunkles UI-Theme fuer die App selbst anbieten.
- [ ] Tooltips fuer neue Form- und Symboloptionen erweitern.
